# BuuttiCodingTask
## Requirements
* [Node.js](https://nodejs.org/)
* [SQLite](https://www.sqlite.org/)
## Installation
* Install Node.js and npm
  ```
  curl -fsSL https://deb.nodesource.com/setup_17.x | sudo -E bash -
  sudo apt-get install -y nodejs
  ```
* Install SQLite
  ```
  npm install -g sqlite --save
  ```
